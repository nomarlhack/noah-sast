### Phase 2: 동적 테스트 (검증)


**테스트 방법:**
1. 직렬화된 데이터를 식별 (쿠키, 요청 본문 등에서 Base64/Hex 인코딩된 데이터)
2. 데이터를 변조하여 재전송
3. 에러 메시지, 응답 변화, 시간 지연으로 역직렬화 발생 확인

**안전한 테스트 페이로드:**
- 시간 기반: `Thread.sleep(5000)` (Java), `time.sleep(5)` (Python) — 응답 지연으로 확인
- 외부 콜백: DNS/HTTP 콜백으로 코드 실행 확인
- 에러 유발: 잘못된 직렬화 데이터로 역직렬화 에러 메시지 확인

**검증 기준:**
- **확인됨**: 동적 테스트로 변조된 직렬화 데이터로 코드 실행(시간 지연, 콜백 수신)이 확인됨
